# VendorDesk — Final MVP Document

---

## App Overview

VendorDesk is a mobile app for small vendors in India who run their business on WhatsApp. The app helps them create quotations fast using AI, manage their leads, and share quotes directly on WhatsApp — without replacing WhatsApp or learning complicated software.

**No login. No authentication. Vendor simply onboards once and starts using the app.**

---

## Tech Stack

| Layer | Choice |
|---|---|
| AI Model | OpenRouter — google/gemini-flash-lite-preview |
| Input Types | Text, Voice, Image |
| Backend | Supabase |
| Database | Supabase |
| Cloud Functions | Supabase |
| Authentication | None in MVP |
| Pricing | Free |

## Screen 3 — Catalogue

### Purpose
Vendor builds their product catalogue here. The AI uses this catalogue when creating quotations. This screen is also accessible anytime from the bottom tab.

### Elements
- Screen heading: "Catalogue"
- Search bar (search products)
- Product list — each product card shows:
  - Product name
  - Price
  - Unit
  - Edit icon
  - Delete icon
- Empty state message when no products added yet
- Add Product button — plus icon (bottom right, floating)

### Add Product — 3 Options (shown as tabs or toggle when plus is tapped)

**Option 1 — Manual Entry (Primary / Default)**
- Product name field
- Price field
- Unit field (e.g. kg, piece, litre, box)
- Save button

**Option 2 — Upload Image**
- Upload image button
- AI reads the image and extracts product details
- Extracted items shown in a review list
- Vendor can edit each item before saving
- Save All button

**Option 3 — Upload Excel**
- Upload Excel file button
- Items extracted and shown in review list
- Vendor can edit each item before saving
- Save All button

### Part B — AI Quotation Workspace

### Purpose
Chat-type interface where the vendor talks to the AI to create a quotation. The vendor is not chatting with the customer — only with the AI. Input can be text or voice.

### Elements
- Back button
- Lead name at top
- Lead WhatsApp number below name
- Chat timeline showing the conversation with AI
- AI message bubbles (left side)
- Vendor message bubbles (right side)
- Input bar at bottom:
  - Text input field: "Type your requirement..."
  - Voice button — tap to speak
- Send button

### How the AI Flow Works

```
AI opens with:
"Hi! Tell me what products you want 
to add in this quotation."

Vendor sends (text or voice):
"5 kg rice 10 ltr milk"

AI checks catalogue:
  ✓ Rice found → adds to quotation
  ✗ Milk not found → AI replies:
    "Milk is not in your catalogue."
    [ Add to Catalogue ]
    [ Just Add to Quotation ]

Vendor taps one of the 2 buttons.

AI continues processing remaining items.

Once all items are processed AI replies:
"Here is your quotation."
[ View Quotation Preview ]
```

- Lead name
- Date (auto)
- Quotation number (auto-generated)

**Editable item table:**
- Item name
- Quantity
- Unit
- Rate
- Line total
- Edit icon on each row
- Delete icon on each row
- Add item row button

**Summary section:**
- Subtotal

- Create PDF button
